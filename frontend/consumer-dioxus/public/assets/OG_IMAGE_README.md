# Default Open Graph Image

## Required File

Create a file named `og-default.png` in this directory.

## Specifications

- **Dimensions**: 1200x630 pixels (recommended by Facebook/Twitter)
- **Format**: PNG or JPG
- **File size**: Under 1MB for optimal loading
- **Content**: Should include:
  - Site name: "Hmziq.rs Blog"
  - Tagline: "Thoughts on software and technology"
  - Branding elements (logo, colors)
  - Clean, professional design

## Purpose

This image is used as the fallback Open Graph image when:
- A blog post doesn't have a featured image
- Static pages (about, contact, etc.) are shared on social media
- The homepage is shared

## Design Tips

1. Use a simple, clear design that represents your brand
2. Ensure text is large enough to read in small previews
3. Use high contrast between background and text
4. Test on multiple devices and platforms
5. Consider dark/light mode users

## Testing

After creating the image, test it with:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
